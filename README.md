# PR0xe 🚀

**Simple & working HTTP/HTTPS proxy for Vercel** — perfect for school computers.

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FGLAXYIAS%2FPR0xe)

## Usage

**With query param:**
`https://your-project.vercel.app/api/proxy?url=https://youtube.com`

**Clean URL:**
`https://your-project.vercel.app/api/proxy/https://youtube.com`

Works with most sites (YouTube, Google, etc. — some heavy JS sites may have issues).

## Files
- `api/proxy.js` — the main proxy
- `vercel.json` — routing config

Enjoy!
